# Master Bullet Library

This file is your personal bullet bank — a growing collection of your best resume bullets,
organized by function and searchable by tag.

The skill uses this file during Phase 2A to source the most relevant bullets for each job
you apply to. The more bullets you add here, the better Phase 2A gets at finding the right
proof points for each role.

---

## How to Use This File

**Add bullets as you accumulate them.** Anytime you rewrite a bullet in Phase 1 or Phase 2,
copy standout versions here. After a few applications, you'll have a library that covers your
career from multiple angles.

**Tag each bullet** with the functional area it best demonstrates. Use the existing tags
or create your own. Tags are what Phase 2A uses to find the right bullets for a given JD.

**Keep bullets in XYZ format:**
> Accomplished [X] as measured by [Y] by doing [Z]

Every bullet should have at least one metric. If you don't have a number yet, add `[verify]`
and come back to it.

**Bullets here are drafts.** Phase 2A will rewrite library bullets to match the current
canonical voice before using them. Don't worry about perfect polish — focus on capturing
the achievement and the metric.

---

## Bullet Format

```markdown
### [FUNCTIONAL-TAG]

- [Bullet text with metric] — source: [Role Title, Company, approx. year]
- [Bullet text with metric] — source: [Role Title, Company, approx. year]
```

## Tags (Customize These for Your Career)

Use these suggested tags or replace them with the functional areas that describe your work.
The skill will match tags against JD keywords to surface the most relevant bullets.

Suggested tags:
- `OPERATIONS` — process design, efficiency, systems
- `REVENUE` — pipeline, sales, growth metrics
- `GTM` — launches, campaigns, go-to-market
- `PRODUCT` — roadmap, delivery, shipping
- `PROGRAM` — cross-functional coordination, delivery management
- `PEOPLE` — hiring, team building, coaching
- `FINANCE` — budgets, cost control, contracts
- `BUSINESS-AFFAIRS` — deals, licensing, negotiations
- `STRATEGY` — planning, market analysis, decision-making
- `TECHNOLOGY` — tools, platforms, technical implementation
- `PARTNERSHIPS` — external relationships, vendors, integrations
- `COMMS` — writing, stakeholder communication, presentations

## Maintenance Tips

- After each Phase 1 rewrite, scan the output for bullets with strong metrics and paste them here
- If a bullet gets used across multiple applications and always lands well, flag it with ⭐
- Remove bullets that no longer reflect your current scope or seniority level
- Review and prune annually — a stale library is worse than a small one
